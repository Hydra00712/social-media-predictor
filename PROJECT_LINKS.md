## üìä Azure Monitoring Queries (for screenshots)

### Sample KQL Queries to Run in Log Analytics:

#### 1. Show Recent Predictions
```kql
AzureQueue
| where TimeGenerated > ago(24h)
| where QueueName == "predictions-queue"
| project TimeGenerated, Message
| order by TimeGenerated desc
| limit 50
```

#### 2. Application Insights - Events
```kql
customEvents
| where name == "PredictionMade"
| project timestamp, name, customDimensions
| order by timestamp desc
| limit 100
```

#### 3. Performance Metrics
```kql
performanceCounters
| where TimeGenerated > ago(1h)
| summarize avg(value) by name, bin(TimeGenerated, 5m)
| render timechart
```
