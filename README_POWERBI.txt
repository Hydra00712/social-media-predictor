POWER BI DATA PACKAGE
======================

FILES INCLUDED:
1. social_media_cleaned.csv - 12,000 social media posts with engagement data
2. social_media.db - Database with live predictions from the app

HOW TO USE:
1. Open Power BI Desktop
2. Get Data -> Text/CSV
3. Load: social_media_cleaned.csv
4. Start creating visualizations

KEY COLUMNS:
- engagement_rate: How many people engaged (0 to 1)
- platform: Instagram, Twitter, Facebook, etc.
- sentiment_score: Positive/negative sentiment (-1 to 1)
- topic_category: Technology, Fashion, Food, etc.
- day_of_week: Monday, Tuesday, etc.

DASHBOARD IDEAS:
- Engagement by Platform (Bar Chart)
- Engagement Over Time (Line Chart)
- Sentiment vs Engagement (Scatter Plot)
- Top Topics (Bar Chart)
- Geographic Map

Good luck!
